# 文档说明与命名规则

本目录存放「电商评论情感与痛点分析系统」相关文档。统一使用英文 **slug**：`ecommerce-review-insights-v1`，与 `{类型}` 后缀组合。

| 目录 / 文件 | 类型 | 说明 |
|-------------|------|------|
| **prd/** | 产品需求 | `ecommerce-review-insights-v1-prd.md` |
| **specs/** | 功能规格 | `ecommerce-review-insights-v1-spec.md` |
| **plans/** | 实施与毕设计划 | `ecommerce-review-insights-v1-plan.md`（含工程里程碑 M0–M3、毕设周次 W0–W5、论文章节建议） |
| **tasks/** | 任务拆解 | `ecommerce-review-insights-v1-task.md` |
| **stage-a/** | Stage A 产物 | `ecommerce-review-insights-v1-ta1-label-eval-freeze.md`、`ecommerce-review-insights-v1-ta2-data-contract.md` |

**链路**：PRD → Spec → Plan → Task；仓库根目录 `AGENTS.md` 供 Agent 技能表引用，不属于产品文档链路。
